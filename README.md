# Proyecto Minicode

**Lenguaje educativo en español** con soporte para:

- Variables, funciones, condicionales y bucles
- Comandos gráficos y musicales
- Polinomios simbólicos con SymPy y graficación
- Ejecución en GUI con PyQt6

---

## 1. Requisitos

- Python 3.11 o superior  
- Java JDK 8+ (para ANTLR4)  

### 1.1 Dependencias Python

Se recomienda usar un entorno virtual. Contenido de `requirements.txt`:

```txt
PyQt6
antlr4-python3-runtime
matplotlib
sympy
numpy
